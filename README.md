# Metahash PHP

## Requirements

#### Basic requirements
- PHP 7.1+
- ext-gmp
- ext-curl
- composer

## Usage

```shell
git clone git@github.com:xboston/php-metahash.git
cd php-metahash
composer install --no-dev
php example/cli.php method=generate
```
